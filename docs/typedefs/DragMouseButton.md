# DragMouseButton

A constant type with available string values
which can be used when configurating the drag behaviour
of a certain graphical element.

- - -

```ts
type DragMouseButton = "left" | "middle" | "right";
```

## Additional details

| Value                  | Type   |
|------------------------|--------|
| ``"left"``          | String |
| ``"middle"``          | String |
| ``"right"``          | String |

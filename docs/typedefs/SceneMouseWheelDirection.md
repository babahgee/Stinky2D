# SceneMouseWheelDirection

A constant type with available string values
which describes the direction of the user mousewheel.

- - -

```ts
type SceneMouseWheelDirection = "up" | "down" | "left" | "right" | null;
```

## Additional details

| Value                | Type     |
|----------------------|----------|
| ``"up"``             | String   |
| ``"down"``           | String   |
| ``"left"``           | String   |
| ``"right"``          | String   |
| ``null``             | ``null`` |
